# Corporate Landing Page

架空のコーポレートサイトを想定したランディングページです。  
HTML / CSS / JavaScriptで構成されています。

CSS変数を初めて導入し、色・余白・フォントサイズを統一することで、再利用しやすい設計を意識しました。

---

## 使用技術
- HTML
- CSS
- JavaScript

---

## 工夫した点
- CSS変数を用いたデザインシステム化
- 色・余白・フォントサイズを統一し、保守性を意識した設計
- レスポンシブ対応によるスマホ最適化
